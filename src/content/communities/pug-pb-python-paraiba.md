PUG-PB (Python User Group Paraíba) é a comunidade de usuários Python no estado. Bastante ativa, ela promove discussões online, meetups periódicos e participa de eventos maiores (como o GrudePB Fest), sempre incentivando a troca de conhecimento e a colaboração em projetos Python entre desenvolvedores paraibanos.
