CEJUG (Ceará Java User Group) é a tradicional comunidade de desenvolvedores Java no Ceará. Apesar do nome, o grupo tem perfil poliglota e abrange diversas tecnologias do ecossistema JVM. A CEJUG promove encontros para compartilhar conhecimento, discutir novidades da plataforma Java e fortalecer a rede de desenvolvedores no estado.
