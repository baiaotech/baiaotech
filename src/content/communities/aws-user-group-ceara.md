AWS User Group Ceará é a comunidade de usuários dos serviços de nuvem AWS no estado. O grupo reúne entusiastas e profissionais de cloud para trocar experiências, estudar boas práticas e organizar eventos (como meetups via o programa AWS User Group), impulsionando o conhecimento em soluções AWS na região.
