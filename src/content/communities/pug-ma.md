PUG-MA (Python User Group Maranhão) é a comunidade Python do estado. Com presença em diversos canais (lista de e-mails, Telegram, Facebook), o grupo conecta entusiastas de Python para discutir a linguagem, trocar dicas, organizar meetups e fortalecer a comunidade de desenvolvedores Python no Maranhão.
