JavaCE (Java User Group Ceará), também conhecido como Java[x] CE, é o grupo de usuários Java do estado. A comunidade divulga a linguagem Java e tecnologias da plataforma JVM, organizando meetups, grupos de estudo e eventos onde desenvolvedores compartilham experiências sobre Java, Kotlin, Spring e outras soluções.
