React Recife é a comunidade dos desenvolvedores de React.js e React Native no Recife. Criada para unir e fortalecer o ecossistema React local, a comunidade promove a troca de ideias e organiza eventos e workshops práticos onde desenvolvedores de todos os níveis aprendem juntos e compartilham soluções envolvendo React (tanto para web quanto mobile).
