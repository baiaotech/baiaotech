PUG-SE (Python User Group Sergipe) é a comunidade de usuários Python no estado de Sergipe. Bastante ativa, ela está presente online (site oficial se.python.org.br, grupos no Telegram, LinkedIn etc.) e em eventos locais. A comunidade incentiva o aprendizado de Python, organiza meetups e contribui para projetos open-source, tornando o ecossistema Python sergipano cada vez mais forte.
