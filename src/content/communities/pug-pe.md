PUG-PE (Python User Group Pernambuco) é a comunidade de usuários Python no estado, reconhecida como uma das mais ativas do país. Com forte presença no Telegram, GitHub, Instagram e listas de discussão, o PUG-PE promove meetups, Python Days e colabora com eventos regionais, difundindo a linguagem Python e conectando desenvolvedores pernambucanos em um ambiente inclusivo de aprendizado.
