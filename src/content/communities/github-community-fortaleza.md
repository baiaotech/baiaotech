GitHub Community Fortaleza é a comunidade local de usuários do GitHub no Ceará. O grupo tem o objetivo de difundir boas práticas de controle de versão, colaboração open-source e uso avançado da plataforma GitHub, por meio de workshops, hackathons e encontros de desenvolvedores da região.
